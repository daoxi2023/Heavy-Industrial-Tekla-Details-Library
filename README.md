# Heavy Industrial Steel Connections - TEKLA Structures Library 🏗️

Welcome to the **Heavy Industrial Tekla Details Library**, an open-source repository of standard structural steel connection components built specifically for TEKLA Structures. 

> **🚧 Status: Work in Progress (Constantly Updating)**
> We are currently organizing and uploading our internal library of heavy industrial connections. Star (⭐) and Watch (👀) this repository to get notified when new `.tekla`, `.ifc`, and `.step` files are released!

## 🎯 What to Expect in This Library
This repository is tailored for structural engineers, detailers, and EPC professionals working on heavy industrial buildings, steel mills, and facilities with overhead cranes. Upcoming uploads will include:
* **Crane Girder Connections:** Heavy-duty brackets and tie-backs.
* **Rigid Column Base Plates:** Anchor bolt layouts for high-moment heavy columns.
* **Heavy Bracing Gussets:** Seismic and wind load connections.
* **Portal Frame Knee Joints:** Standardized haunch details.
